# Problem Structure Definition

Problem Name: "Simple Addition"
Function Name: addTwoNumbers

Input Structure:
Input Field: int a
Input Field: int b

Output Structure:
Output Field: int sum

## Constraints:
- -10^9 ≤ a ≤ 10^9
- -10^9 ≤ b ≤ 10^9
- The result fits in a 32-bit signed integer

## Time Limit: 
1 second

## Memory Limit: 
128 MB
